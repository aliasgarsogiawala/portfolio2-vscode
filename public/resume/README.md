# Resume Download

To download my resume, open the terminal and run the following command:

```bash
download resume.pdf
```

This will download my latest resume to your local machine.

## Alternative Commands

You can also use:
- `wget resume.pdf` - Download using wget
- `curl -O resume.pdf` - Download using curl
- `get resume.pdf` - Simple get command

The resume is always kept up-to-date with my latest experience and skills.
